\section{Comparación de estrategias}

En esta sección se comparan los cuatro enfoques y se justifica por qué una misma técnica no sirve igual para todos los casos.

\subsection{Comparación general}
\begin{itemize}
    \item Optimización y búsqueda local: útil cuando se busca mejorar una solución.
    \item Espacios continuos: necesario cuando las variables toman valores reales.
    \item No determinismo: requiere planes condicionales.
    \item Observabilidad parcial: exige razonar con incertidumbre.
\end{itemize}
