\section{Problema 3: Búsqueda con acciones no deterministas}

\subsection{Modelado del problema}
Representar un robot de almacén donde una acción puede producir varios resultados.

\subsubsection{Estado}
Posición del robot, estado de la puerta, carga transportada, etc.

\subsubsection{Acciones}
Avanzar, abrir puerta, girar, recoger objeto.

\subsubsection{Resultados posibles}
Una misma acción puede tener éxito, fallar o producir un resultado intermedio.

\subsubsection{Dificultad principal}
No existe una secuencia fija de acciones que garantice siempre el mismo resultado.

\subsection{Algoritmo o enfoque elegido}
Planes condicionales, planes ramificados o árboles AND-OR.

\subsection{Funcionamiento paso a paso}
Mostrar qué hace el plan si ocurre cada posible resultado.

\subsection{Limitaciones}
Mayor complejidad y necesidad de contemplar múltiples ramas.
